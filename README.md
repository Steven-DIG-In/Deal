# DealIQ by Aspect Advisory

AI-powered deal profitability assessment tool for bank relationship managers and credit analysts.

Built with Next.js · Tailwind CSS · Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Login Credentials (Demo)

Set in `.env.local` (copy from `.env.example`):

```
DEMO_EMAIL=demo@aspectadvisory.eu
DEMO_PASSWORD=DealIQ2025
```

## Deploy to Vercel

1. Push this repo to GitHub
2. Import in Vercel
3. Add environment variables from `.env.example`
4. Deploy

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **httpOnly cookie auth** (upgrade to Supabase Auth in Phase 2)
